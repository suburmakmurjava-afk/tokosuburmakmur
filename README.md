# TokoTaniTernak Pro - Panduan

Aplikasi katalog untuk produk pertanian dan peternakan dengan fitur offline penuh.

## Cara Menjalankan

1. Pastikan Flutter SDK terinstall (versi 3.0+)
2. Jalankan perintah berikut:

```bash
cd TokoTaniTernak_Pro
flutter pub get
flutter run
```

## Cara Build APK

```bash
flutter build apk --release
```

APK akan tersedia di: `build/app/outputs/flutter-apk/app-release.apk`

## Fitur

- ✅ 100% Offline - Semua data tersimpan lokal
- ✅ Database Isar untuk penyimpanan cepat
- ✅ Penyimpanan gambar lokal
- ✅ Kategori: Pertanian & Peternakan
- ✅ Desain Modern Material 3
